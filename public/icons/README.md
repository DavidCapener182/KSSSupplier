# Required Icons

Please add the following icon files to this directory (`public/icons/`):

1. `icon-192x192.png` (192x192 px, PNG) - For Android Home Screen
2. `icon-512x512.png` (512x512 px, PNG) - For Splash Screen / App Store

## Additional Icons Required in `public/` (Root)

Please add these files to the parent directory (`public/`):

1. `apple-touch-icon.png` (180x180 px, PNG) - For iPhone/iPad Home Screen
2. `favicon.ico` (32x32 px, ICO) - For Browser Tab Bar
